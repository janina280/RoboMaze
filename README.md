**RoboMaze** is an interactive educational game designed to teach basic programming logic.  
Players control a robot that must navigate from point A to point B using programmable blocks. The goal is to reach the destination using the most efficient path.

---

## üåü Key Features

### Gameplay
- Control the robot using programming blocks: `Move Up`, `Move Down`, `Move Left`, `Move Right`.   
- Loop blocks: `REPEAT 2 TIMES`. 

### Maze
- Procedurally generated maze for each level.    

### Progress and Scoring
- Tutorial levels guide players gradually.   
- Achievements and badges track player progress.  
- Progress saved in LocalStorage, so the game continues after a refresh.  

### Visual and Audio Feedback
- Robot animations for movement and collisions.  
- Configurable background music and sound effects.  
- Interactive tutorials and hints for each level.  

### Other Features
- Optional timer. 
- Visual code editor and block editor for intuitive programming.  
- Debugger shows execution steps of blocks and robot states.
